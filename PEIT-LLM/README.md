## Files
* `llama3_lora_predict.yaml`: the code of llama3.1 lora prediction.
* `llama3_lora_sft.yaml`: the code of llama3.1 lora sft .

## Requirements
Run `git clone --depth 1 https://github.com/hiyouga/LLaMA-Factory.git cd LLaMA-Factory pip install -e ".[torch,metrics]"` to install the required packages.

## Dataset
Add the instruction data generated by PEIT-GEN to the dataset_info.json file and define it.

## Lora SFT
You should manually add the name of your dataset definition to llama3_lora_sft.yaml, and then start the training.
  * `llamafactory-cli train llama3_lora_sft.yaml`

## Lora Prediction
You should manually add the name of your dataset definition to llama3_lora_predict.yaml, and then start the prediction.
  * `llamafactory-cli train llama3_lora_predict.yaml`
